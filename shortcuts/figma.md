# Figma Shortcuts

## Tools
- `V`: Move Tool
- `F`: Frame Tool
- `R`: Rectangle Tool
- `O`: Oval Tool
- `T`: Text Tool

## Edit
- `Ctrl+D`: Duplicate
- `Ctrl+G`: Group Selection
- `Ctrl+Shift+G`: Ungroup
- `Ctrl+[`: Send Backward
- `Ctrl+]`: Bring Forward

## View
- `Ctrl++`: Zoom In
- `Ctrl+-`: Zoom Out
- `Shift+1`: Zoom to 100%
- `Shift+2`: Zoom to Fit
- `Space`: Pan Tool

## Components
- `Ctrl+Alt+K`: Create Component
- `Ctrl+Shift+K`: Detach Instance
- `Alt+1`: Show/Hide Left Sidebar
- `Alt+2`: Show/Hide Right Sidebar
- `\`: Show Layout Grids
