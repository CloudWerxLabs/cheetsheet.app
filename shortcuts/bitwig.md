# Bitwig Studio Shortcuts

## Transport
- `Space`: Play/Stop
- `Shift+Space`: Continue
- `Shift+Record`: Record
- `Num .`: Stop
- `L`: Loop Toggle

## Edit
- `Ctrl+D`: Duplicate
- `Delete`: Delete Selection
- `Alt+X`: Split
- `Ctrl+J`: Join
- `Ctrl+M`: Merge

## View
- `Tab`: Toggle Detail Editor
- `F1`: Arrange View
- `F2`: Mix View
- `F3`: Edit View
- `F4`: Browser

## Tools
- `1`: Select Tool
- `2`: Pen Tool
- `3`: Eraser Tool
- `4`: Knife Tool
- `5`: Time Tool
