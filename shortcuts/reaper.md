# REAPER Shortcuts

## Transport
- `Space`: Play/Stop
- `Ctrl+R`: Record
- `Home`: Go to Start
- `End`: Go to End
- `Enter`: Go to Time

## Edit
- `S`: Split Items
- `Ctrl+D`: Duplicate Items
- `G`: Group Items
- `U`: Ungroup Items
- `M`: Mute Items

## View
- `F3`: Show Mixer
- `F2`: Show Navigator
- `F11`: Full Screen
- `+`: Zoom In
- `-`: Zoom Out

## Tools
- `1`: Item Tool
- `2`: Time Selection Tool
- `3`: Draw Tool
- `4`: Envelope Tool
- `5`: Stretch Tool
