# Unreal Engine Shortcuts

## Navigation
- `Right Mouse`: Orbit
- `Left Mouse`: Select
- `Middle Mouse`: Pan
- `F`: Focus Selected
- `End`: Snap to Ground

## Transform
- `W`: Translate
- `E`: Rotate
- `R`: Scale
- `Ctrl+L`: Lock Selection
- `Alt+S`: Snap to Surface

## Editor
- `G`: Game View
- `Ctrl+P`: Play in Editor
- `F8`: Eject from PIE
- `Shift+F1`: Content Browser
- `Shift+F2`: Blueprint Editor

## Viewport
- `F11`: Immersive Mode
- `G`: Game View
- `Ctrl+Space`: Maximize Viewport
- `Alt+G`: Hide Selected
- `H`: Hide Selection
