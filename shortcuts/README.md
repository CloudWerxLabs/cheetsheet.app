# Shortcuts Directory

Place markdown files for application shortcuts here.

## Naming Convention
- Use lowercase process name as filename
- Extension must be `.md`

## Example
- `chrome.md`
- `vscode.md`
- `notepad.md`

## Markdown Format
```markdown
# Application Name Shortcuts

## Category 1
- `Ctrl+C`: Copy
- `Ctrl+V`: Paste

## Category 2
- `Alt+F4`: Close Window
```
