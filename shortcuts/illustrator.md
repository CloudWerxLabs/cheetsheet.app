# Adobe Illustrator Shortcuts

## Tools
- `V`: Selection Tool
- `A`: Direct Selection Tool
- `P`: Pen Tool
- `T`: Type Tool
- `M`: Rectangle Tool
- `L`: Ellipse Tool

## Object Manipulation
- `Ctrl+G`: Group
- `Ctrl+Shift+G`: Ungroup
- `Ctrl+[`: Send Backward
- `Ctrl+]`: Bring Forward
- `Ctrl+Y`: Outline View

## Path Operations
- `Ctrl+8`: Compound Path
- `Ctrl+Shift+Alt+O`: Offset Path
- `Ctrl+J`: Join
- `Shift+O`: Reflect Tool
- `R`: Rotate Tool

## View
- `Ctrl++`: Zoom In
- `Ctrl+-`: Zoom Out
- `Ctrl+0`: Fit Artboard
- `F`: Toggle Screen Modes
- `Tab`: Hide/Show Panels
