# GarageBand Shortcuts

## Transport
- `Space`: Play/Stop
- `R`: Record
- `Return`: Go to Beginning
- `Shift+R`: Toggle Cycle
- `K`: Toggle Metronome

## Edit
- `Cmd+Z`: Undo
- `Cmd+C`: Copy
- `Cmd+V`: Paste
- `Cmd+X`: Cut
- `Delete`: Delete

## View
- `Y`: Show/Hide Library
- `E`: Show/Hide Editor
- `B`: Show/Hide Browser
- `I`: Show Track Info
- `M`: Show/Hide Smart Controls

## Tools
- `T`: Track Select
- `P`: Piano Roll
- `A`: Automation
- `F`: Fade Tool
- `G`: Groove Track
