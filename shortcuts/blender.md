# Blender Shortcuts

## Navigation
- `Middle Mouse`: Orbit View
- `Shift+Middle Mouse`: Pan View
- `Scroll`: Zoom View
- `Numpad 1`: Front View
- `Numpad 7`: Top View

## Selection
- `A`: Select/Deselect All
- `B`: Box Select
- `C`: Circle Select
- `Alt+Click`: Edge Loop
- `L`: Select Linked

## Modeling
- `E`: Extrude
- `S`: Scale
- `R`: Rotate
- `G`: Grab/Move
- `X`: Delete

## Mode Switching
- `Tab`: Edit/Object Mode
- `Ctrl+Tab`: Mode Pie Menu
- `Shift+A`: Add Menu
- `Shift+D`: Duplicate
- `Alt+D`: Linked Duplicate
