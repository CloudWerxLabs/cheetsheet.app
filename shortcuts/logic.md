# Logic Pro Shortcuts

## Transport
- `Space`: Play/Stop
- `R`: Record
- `Return`: Go to Beginning
- `Enter`: Go to Selection Start
- `.`: Stop Recording

## Edit
- `Cmd+Z`: Undo
- `Cmd+C`: Copy
- `Cmd+V`: Paste
- `Cmd+X`: Cut
- `Delete`: Delete

## Tools
- `T`: Pointer Tool
- `C`: Pencil Tool
- `D`: Drum Editor
- `P`: Piano Roll
- `S`: Score Editor

## Windows
- `X`: Mixer
- `B`: Smart Controls
- `Y`: Library
- `E`: Editor
- `L`: Loop Browser
