# AutoCAD Shortcuts

## Drawing
- `L`: Line
- `C`: Circle
- `A`: Arc
- `REC`: Rectangle
- `PL`: Polyline

## Editing
- `CO`: Copy
- `M`: Move
- `RO`: Rotate
- `TR`: Trim
- `E`: Erase

## View
- `Z`: Zoom
- `P`: Pan
- `V`: View
- `RE`: Regenerate
- `F`: Fit to Screen

## Layers
- `LA`: Layer Properties
- `LT`: Layer Tools
- `LM`: Layer Match
- `LF`: Layer Freeze
- `LO`: Layer Off
