# Pro Tools Shortcuts

## Transport
- `Space`: Play/Stop
- `F12`: Record
- `Return`: Go to Beginning
- `End`: Go to End
- `Ctrl+Space`: Pre-Roll

## Edit
- `B`: Smart Tool
- `A`: Selector Tool
- `F`: Hand Tool
- `T`: Trim Tool
- `P`: Pencil Tool

## View
- `Alt+[`: Zoom Out Horizontally
- `Alt+]`: Zoom In Horizontally
- `Alt+=`: Zoom In Vertically
- `Alt+-`: Zoom Out Vertically
- `F`: Fit Tracks to Window

## Windows
- `Ctrl+1`: Mix Window
- `Ctrl+2`: Edit Window
- `Ctrl+3`: Transport Window
- `Ctrl+4`: Memory Locations
- `Ctrl+5`: MIDI Editor
