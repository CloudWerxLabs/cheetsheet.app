# Sketch Shortcuts

## Tools
- `V`: Move Tool
- `R`: Rectangle
- `O`: Oval
- `L`: Line
- `T`: Text

## Editing
- `Cmd+D`: Duplicate
- `Cmd+G`: Group
- `Cmd+Shift+G`: Ungroup
- `Alt+Cmd+C`: Copy Style
- `Alt+Cmd+V`: Paste Style

## View
- `Cmd+1`: Zoom to 100%
- `Cmd+2`: Zoom to Selection
- `Cmd+3`: Center Canvas
- `Space`: Pan Canvas
- `Tab`: Toggle Sidebar

## Layers
- `Cmd+R`: Rename Layer
- `Cmd+F`: Find Layer
- `Cmd+Up`: Bring Forward
- `Cmd+Down`: Send Backward
- `Cmd+Alt+Up`: Bring to Front
