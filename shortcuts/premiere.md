# Adobe Premiere Pro Shortcuts

## Timeline
- `Space`: Play/Pause
- `J`: Rewind
- `K`: Stop
- `L`: Forward
- `I`: Mark In
- `O`: Mark Out

## Editing
- `C`: Razor Tool
- `V`: Selection Tool
- `B`: Ripple Edit Tool
- `N`: Rolling Edit Tool
- `Ctrl+K`: Split Clip

## Tools
- `A`: Track Select Tool
- `T`: Rate Stretch Tool
- `P`: Pen Tool
- `H`: Hand Tool
- `Z`: Zoom Tool

## Panels
- `Shift+1`: Project Panel
- `Shift+2`: Source Monitor
- `Shift+3`: Timeline Panel
- `Shift+4`: Program Monitor
- `~`: Maximize Panel
