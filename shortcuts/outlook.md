# Microsoft Outlook Shortcuts

## Mail
- `Ctrl+N`: New Email
- `Ctrl+R`: Reply
- `Ctrl+Shift+R`: Reply All
- `Ctrl+F`: Forward
- `Ctrl+Enter`: Send

## Navigation
- `Ctrl+1`: Mail
- `Ctrl+2`: Calendar
- `Ctrl+3`: Contacts
- `Ctrl+4`: Tasks
- `Ctrl+Y`: Go to Folder

## Organization
- `Ctrl+Shift+V`: Move to Folder
- `Ctrl+Shift+I`: Go to Inbox
- `Ctrl+E`: Search
- `Insert`: Flag Message
- `Delete`: Delete Item

## Calendar
- `Ctrl+Shift+A`: New Appointment
- `Ctrl+Shift+Q`: New Meeting
- `Alt+1`: Day View
- `Alt+2`: Week View
- `Alt+3`: Month View
