# Adobe Audition Shortcuts

## Transport
- `Space`: Play/Stop
- `Shift+Space`: Play from In to Out
- `Home`: Return to Start
- `End`: Go to End
- `.`: Go to Next Marker

## Edit
- `Ctrl+C`: Copy
- `Ctrl+X`: Cut
- `Ctrl+V`: Paste
- `Delete`: Delete Selected
- `Ctrl+Z`: Undo

## View
- `\\`: Zoom to Selection
- `[`: Zoom Out Full
- `]`: Zoom In Full
- `Ctrl+[`: Decrease Height
- `Ctrl+]`: Increase Height

## Tools
- `R`: Time Selection
- `P`: Paint Tool
- `B`: Brush Selection
- `H`: Hand Tool
- `Z`: Zoom Tool
