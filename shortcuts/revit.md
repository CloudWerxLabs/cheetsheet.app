# Revit Shortcuts

## Drawing
- `WA`: Wall
- `DR`: Door
- `WI`: Window
- `CO`: Component
- `RM`: Room

## Modification
- `AL`: Align
- `TR`: Trim
- `OF`: Offset
- `AR`: Array
- `MI`: Mirror

## View
- `VV`: View Properties
- `VG`: Visibility Graphics
- `ZF`: Zoom to Fit
- `ZP`: Zoom Previous
- `VR`: View Range

## Selection
- `WT`: Wall Types
- `FT`: Filter
- `TL`: Thin Lines
- `HH`: Hide Elements
- `IS`: Isolate Elements
