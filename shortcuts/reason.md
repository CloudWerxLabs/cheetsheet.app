# Reason Shortcuts

## Transport
- `Space`: Play/Stop
- `Num *`: Record
- `Return`: Go to Left Locator
- `Shift+Return`: Go to Right Locator
- `Q`: Quantize

## Edit
- `Ctrl+C`: Copy
- `Ctrl+X`: Cut
- `Ctrl+V`: Paste
- `Delete`: Delete
- `Ctrl+D`: Duplicate

## View
- `F5`: Sequencer
- `F6`: Rack
- `F7`: Mixer
- `F8`: Tools Window
- `Tab`: Toggle Rack/Sequencer

## Tools
- `Q`: Arrow Tool
- `W`: Pencil Tool
- `E`: Eraser Tool
- `R`: Razor Tool
- `T`: Mute Tool
