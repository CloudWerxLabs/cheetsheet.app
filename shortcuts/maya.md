# Maya Shortcuts

## Navigation
- `Alt+LMB`: Rotate View
- `Alt+MMB`: Move View
- `Alt+RMB`: Zoom View
- `F`: Frame Selection
- `A`: Frame All

## Selection
- `RMB`: Selection Menu
- `Ctrl+A`: Select All
- `Shift+A`: Deselect All
- `>`: Select Next Object
- `<`: Select Previous Object

## Transform
- `W`: Move Tool
- `E`: Rotate Tool
- `R`: Scale Tool
- `Insert`: Create New Layer
- `D`: Modify Pivot

## Display
- `4`: Wireframe
- `5`: Shaded
- `6`: Textured
- `7`: Use All Lights
- `8`: Smooth Shade All
