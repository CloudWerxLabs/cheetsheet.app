# Studio One Shortcuts

## Transport
- `Space`: Play/Stop
- `Num *`: Record
- `Return`: Return to Zero
- `Num .`: Stop
- `L`: Loop Active

## Edit
- `X`: Split at Cursor
- `Ctrl+D`: Duplicate
- `Ctrl+B`: Bounce Selection
- `Ctrl+M`: Merge Events
- `Alt+X`: Split at Grid

## View
- `F3`: Console
- `F4`: Inspector
- `F5`: Browser
- `F7`: Show Effects
- `E`: Show Editor

## Tools
- `1`: Arrow Tool
- `2`: Range Tool
- `3`: Split Tool
- `4`: Eraser Tool
- `5`: Paint Tool
