# 3ds Max Shortcuts

## Navigation
- `Alt+W`: Move View
- `Alt+E`: Rotate View
- `Alt+R`: Zoom View
- `F3`: Wire Frame
- `F4`: Smooth Shading

## Transform
- `W`: Select and Move
- `E`: Select and Rotate
- `R`: Select and Scale
- `Q`: Select and Link
- `A`: Angle Snap Toggle

## Modeling
- `Shift+A`: Create Box
- `Shift+S`: Create Sphere
- `Shift+C`: Create Cylinder
- `Shift+L`: Create Line
- `Shift+P`: Create Plane

## Display
- `F2`: Show Grid
- `F3`: Wire Frame
- `F4`: Edged Faces
- `G`: Grid Snap Toggle
- `S`: Snap Toggle
