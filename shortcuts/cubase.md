# Cubase Shortcuts

## Transport
- `Space`: Start/Stop
- `Num *`: Record
- `Num 1`: Go to Left Locator
- `Num 2`: Go to Right Locator
- `Num 7`: Return to Zero

## Edit
- `X`: Split at Cursor
- `Delete`: Delete Selected
- `Ctrl+D`: Duplicate
- `Alt+D`: Repeat
- `U`: Unmute Objects

## Tools
- `1`: Object Selection
- `2`: Range Selection
- `3`: Draw
- `4`: Erase
- `5`: Split

## Windows
- `F3`: Mixer
- `F4`: VST Instruments
- `F5`: Pool
- `F7`: Transport Panel
- `F8`: Audio Connections
