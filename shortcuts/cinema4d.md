# Cinema 4D Shortcuts

## Navigation
- `1`: Front View
- `2`: Right View
- `3`: Top View
- `4`: Perspective View
- `5`: Back View

## Tools
- `E`: Move
- `R`: Scale
- `T`: Rotate
- `D`: Model Mode
- `F`: Frame Selected

## Modeling
- `C`: Make Editable
- `U`: Subdivide
- `I`: Inner Extrude
- `M`: Extrude
- `B`: Bridge Tool

## Display
- `Shift+V`: Toggle Visibility
- `Shift+S`: Solo Mode
- `O`: Object Manager
- `H`: Hide Selected
- `Alt+H`: Show All
