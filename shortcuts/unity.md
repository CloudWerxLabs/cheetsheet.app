# Unity Shortcuts

## Scene Navigation
- `Q`: Pan Tool
- `W`: Move Tool
- `E`: Rotate Tool
- `R`: Scale Tool
- `F`: Frame Selected

## Game Controls
- `Ctrl+P`: Play/Stop
- `Ctrl+Shift+P`: Pause
- `Ctrl+Alt+P`: Step
- `Shift+Space`: Maximize Window
- `Ctrl+Shift+C`: Enter Play Mode

## Window Management
- `Ctrl+1`: Scene View
- `Ctrl+2`: Game View
- `Ctrl+3`: Inspector
- `Ctrl+4`: Hierarchy
- `Ctrl+5`: Project Browser

## Edit
- `Ctrl+D`: Duplicate
- `F2`: Rename
- `Delete`: Delete
- `Ctrl+S`: Save Scene
- `Ctrl+N`: New Scene
